### What I learned

#### BEM
BEM is a naming convention for CSS class names. It consists of:
  - Block (Component)
  - Element (Child)
  - Modifier

```css
[block]__[element]--[modifier] {

}

.parent {
  /* ... */
}

.parent__header {
  /* ... */
}

.parent__description {
  /* ... */
}

.parent__img {
  /* ... */
}

.btn {
  /* ... */
}

.btn--primary {
  /* ... */
}
```
